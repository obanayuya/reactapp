　npm i -D webpack

↓以下、使うことが予想されるコマンド

◎bundle.jsの更新
	下記コマンドラインをコマンドプロンプトで実行↓
	npm run builds

◎bundle.jsの自動更新
	下記コマンドラインを実行↓
	npm run watch

◎reactサーバーの起動方法
	エクリプスで　/src/main/java/src/main/java/App.javaを右クリック
　　　　実行(run)　→　java アプリケーションをクリック
	トムキャットがインストールされていれば正常に動き出します
